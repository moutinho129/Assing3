Detailed Description

Ficheiro para teste dos comandos do sistema SETR Mini-Projeto.

Este ficheiro contém a implementação das funções que foram declaradas no ficheiro mock_cmd.h. Ficheiro identico ao ficheiro cmd.c mas usado para Unit Testing.
